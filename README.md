# blender-ymd-io-fixed
A Blender plugin that decompiles Yo-kai Watch Puni Puni ".ez" files and convert ".ymd" files to Blender.
- **Do not use it for collaborative models (or share it) because of copyright issues.**
- **Please use common sense**

## How to install?
On Blender, go to "Edit" --> "Preferences..." --> "Add-ons" --> "Install from disk..." and open the ".zip" you downloaded on the Releases.

To install "pycryptodome", go to:
```bash
C:\Program Files\Blender Foundation\Your Blender Version\Your Blender Version\python\bin"
````
Then open CMD and write:
```bash
./python.exe -m pip install pycryptodome
````
## Requirements
- Any Blender v4 instalation
- pycryptodome
- At least Python 3: https://www.python.org/downloads/
